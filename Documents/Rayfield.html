Rayfield UI Library Documentation

A comprehensive guide to using the Rayfield UI Library for Roblox development.

Introduction

Rayfield is a customizable UI library for Roblox that provides a variety of components such as buttons, sliders, and checkboxes. This documentation will guide you through using the library and integrating it into your projects.

Components

  • Toggle Button: A button that toggles between two states (On/Off).
  • Slider: A slider for selecting values within a range.
  • Checkbox: A simple checkbox for binary options.

Installation

Follow the steps below to install Rayfield into your Roblox project:

  1. Download the Rayfield library from GitHub.
  2. Import the library into your Roblox Studio.
  3. Follow the setup instructions in the README file.

Usage

Here are some examples of how to use Rayfield components in your game:


-- Example of creating a Toggle Button
local toggle = Rayfield:CreateToggle({
    Title = "Enable Feature",
    Default = false,
    Callback = function(value)
        print("Toggle value:", value)
    end
})
